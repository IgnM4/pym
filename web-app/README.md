# Web App

Interfaz web estática para visualizar KPIs y realizar ventas simples en un entorno POS. Puede abrirse directamente en un navegador en cualquier dispositivo.

## Uso

1. Abrir `public/index.html` para ver los KPIs de ventas.
2. Desde esa página se puede acceder al módulo **POS** para simular ventas.

El módulo POS carga un listado de productos desde `data/products.json` y permite agregar ítems a un carrito y procesar una venta de manera local (no persiste en servidor).
